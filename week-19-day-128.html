Asking Critical Questions About Sources

Day 127 taught you to share news without sensationalism. Day 128 teaches you how to ask critical questions about sources — evaluating credibility, checking facts, detecting bias, and evaluating evidence. B2 speakers accept information without questioning. C1 speakers ask critical questions, evaluate sources critically, and think critically about information.

C1 Insight
B2 Speaker: Accepts information without questioning. Says "I read this somewhere" or "I saw this on social media." Doesn't evaluate sources or think critically.

C1 Speaker: Asks critical questions, evaluates sources critically, checks facts, detects bias, and evaluates evidence. Asks "Who wrote this?" and "What's the source?" and "What's the evidence?"

Today's Philosophy

Asking Critical Questions = Identifying Source + Checking Credibility + Verifying Facts + Detecting Bias + Evaluating Evidence

"Asking critical questions about sources isn't about being skeptical. It's about identifying the source, checking credibility, verifying facts, detecting bias, and evaluating evidence to think critically about information."

Every source evaluation requires you to make five decisions simultaneously:

Source Decision: How do I identify the source? (Vague? Specific?)
Credibility Decision: How do I check credibility? (Accept? Question?)
Fact Decision: How do I verify facts? (Accept? Verify?)
Bias Decision: How do I detect bias? (Ignore? Detect?)
Evidence Decision: How do I evaluate evidence? (Accept? Evaluate?)

Important: Day 128 teaches you how to ask critical questions about sources with skepticism, critical thinking, and critical evaluation.

The Critical Question Loop

The 5-step process for asking critical questions about sources

1
IDENTIFY SOURCE (Identify the source)
How do I identify? "Who wrote this?" / "What's the source?" (Identify specifically)
2
CHECK CREDIBILITY (Check credibility)
How do I check? "Is this source credible?" / "Is this a reliable source?" (Check critically)
3
VERIFY FACTS (Verify facts)
How do I verify? "What's the evidence?" / "What's the evidence for this claim?" (Verify critically)
4
DETECT BIAS (Detect bias)
How do I detect? "Is there any bias?" / "Is this source biased?" (Detect critically)
5
EVALUATE EVIDENCE (Evaluate evidence)
How do I evaluate? "What's the evidence for this?" / "What's the evidence supporting this?" (Evaluate critically)

3 Common Mistakes

Traps that destroy critical questioning

Mistake 1: Accepting Without Questioning

❌ "I read this somewhere."
✅ "Who wrote this? What's the source? I'd like to know who wrote this and what the source is."
Why: Accepting without questioning prevents critical thinking. Questioning helps you think critically.

Mistake 2: Trusting Single Source

❌ "I saw this on social media."
✅ "Is this source credible? Is this a reliable source? I'd like to check if this is a credible source."
Why: Trusting single source prevents critical thinking. Checking credibility helps you think critically.

Mistake 3: Ignoring Bias

❌ "Is this biased?"
✅ "Is there any bias in this source? Is this source biased? I'd like to detect any bias in this source."
Why: Ignoring bias prevents critical thinking. Detecting bias helps you think critically.

Micro Writing: Critical Question Messages (with Model Answers)

✍️ 4 Writing Tasks with Model Answers

Task 1: News Article Evaluation

Context: You read a news article and want to evaluate it critically. Write a message evaluating it critically.

Requirements:
  • Use 3 collocations from this lesson
  • Use 2 source identification structures (e.g., "Who wrote this?", "What's the source?")
  • Use 2 credibility check structures (e.g., "Is this source credible?", "Is this a reliable source?")
  • Length: 100-150 words
Model Answer (C1)

"I read this news article. Who wrote this? What's the source? I'd like to know who wrote this and what the source is. Is this source credible? Is this a reliable source? I'd like to check if this is a credible source. What's the evidence for this claim? What's the evidence supporting this? I'd like to verify the facts. Is there any bias in this source? Is this source biased? I'd like to detect any bias in this source. What's the evidence for this? What's the evidence supporting this? I'd like to evaluate the evidence."

Language Breakdown
  • Source identification: "Who wrote this?", "What's the source?"
  • Collocations: "source credible", "reliable source", "verify the facts"
  • Credibility check: "Is this source credible?", "Is this a reliable source?"
  • Cohesion markers: "Who wrote this", "What's the source", "I'd like to know"
Alternative Version (Casual)

"I read this news article. Who wrote this? What's the source? I'd like to know who wrote this and what the source is. Is this source credible? Is this a reliable source? I'd like to check if this is a credible source. What's the evidence for this claim? What's the evidence supporting this? I'd like to verify the facts."

Self-check Checklist
  • □ đủ word count (100-150 words)
  • □ dùng đủ 3 collocations (source credible, reliable source, verify the facts)
  • □ dùng đủ 2 source identification structures (Who wrote this, What's the source)
  • □ dùng đủ 2 credibility check structures (Is this source credible, Is this a reliable source)
  • □ có opening → development → closing

Strategic Grammar in Context (5 C1 Patterns)

📝 Strategic Grammar Toolkit (5 C1 Patterns with explanation, examples, common mistakes, native alternatives)

Pattern 1: Source Identification with "Who wrote this"

Used to identify source specifically. These structures show critical thinking.
  • "Who wrote this?"
  • "What's the source?"
  • "I'd like to know who wrote this and what the source is."
  • "Who wrote this article?"
  • "What's the source of this information?"
Common Mistake: "I read this somewhere." (Vague)
Native Alternative: "Who wrote this? What's the source?"

Pattern 2: Credibility Check with "Is this source credible"

Used to check credibility critically. These structures show critical thinking.
  • "Is this source credible?"
  • "Is this a reliable source?"
  • "I'd like to check if this is a credible source."
  • "Is this source reliable?"
  • "Is this a credible source?"
Common Mistake: "Is this true?" (Too simple)
Native Alternative: "Is this source credible?"

Pattern 3: Fact Verification with "What's the evidence for this claim"

  • "What's the evidence supporting this?"

  • "What's the evidence for this?"
  • "What's the evidence supporting this claim?"
Common Mistake: "Is this true?" (Too simple)
