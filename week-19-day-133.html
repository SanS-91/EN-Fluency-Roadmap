Check an Opposing News Source

Days 127-132 taught you to discuss current events, debate social issues, talk to kids about fake news, discuss politics with people who disagree, navigate political disagreements, and handle travel disruptions. Day 133 teaches you how to check an opposing news source — verifying information from different perspectives, cross-referencing sources, and evaluating credibility. B2 speakers accept one source or dismiss opposing views. C1 speakers check multiple sources, evaluate credibility, and form balanced opinions.

Today's Philosophy

Checking Sources = Verifying + Cross-referencing + Evaluating + Forming Opinions

"Checking opposing news sources isn't about finding the 'right' answer. It's about verifying information, cross-referencing sources, evaluating credibility, and forming balanced opinions."

Every news verification requires you to make five decisions simultaneously:

Verification Decision: How do I verify? (Accept? Verify?)
Cross-reference Decision: How do I cross-reference? (One source? Multiple?)
Evaluation Decision: How do I evaluate credibility? (Accept? Evaluate?)
Opinion Decision: How do I form opinions? (Biased? Balanced?)
Bias Decision: How do I recognize bias? (Unaware? Aware?)

Important: Day 133 teaches you how to check opposing news sources with verification, cross-referencing, and critical thinking.

The Source Verification Loop

The 4-step process for checking opposing news sources

1
VERIFY (Verify information)
How do I verify? "Let me verify this information..." / "Let me check if this is accurate..." (Verify thoroughly)
2
CROSS-REFERENCE (Cross-reference sources)
How do I cross-reference? "Let me check multiple sources..." / "Let me cross-reference this with other sources..." (Cross-reference comprehensively)
3
EVALUATE (Evaluate credibility)
How do I evaluate? "Let me evaluate the credibility of this source..." / "Let me evaluate the credibility..." (Evaluate critically)
4
FORM OPINION (Form balanced opinion)
How do I form opinions? "Based on multiple sources..." / "Based on my evaluation..." (Form balanced opinions)

3 Common Mistakes

Traps that destroy source verification

Mistake 1: Accepting Without Verifying

❌ "I read it in the news, so it must be true."
✅ "Let me verify this information by checking multiple sources. I want to make sure this is accurate."
Why: Accepting without verifying may lead to misinformation. Verifying ensures accuracy.

Mistake 2: Using One Source

❌ "I read it in one source, so it's true."
✅ "Let me cross-reference this with multiple sources to get a balanced perspective."
Why: Using one source may lead to biased information. Cross-referencing gets balanced perspectives.

Mistake 3: Forming Biased Opinions

❌ "I think this is true because I agree with it."
✅ "Based on multiple sources and my evaluation, I think... I want to form a balanced opinion."
Why: Forming biased opinions leads to uninformed opinions. Forming balanced opinions shows critical thinking.

Micro Writing: Source Verification Messages (with Model Answers)

✍️ 4 Writing Tasks with Model Answers

Task 1: News Article Verification

Context: You read a news article and want to verify the information. Write a message verifying the information.

Requirements:
  • Use 3 collocations from this lesson
  • Use 2 verification structures (e.g., "Let me verify this information...", "I want to make sure...")
  • Use 2 evaluation structures (e.g., "Let me evaluate the credibility...", "I want to check...")
  • Length: 100-150 words
Model Answer (C1)

"I read this news article and I want to verify the information. Let me verify this information by checking multiple sources. I want to make sure this is accurate before forming an opinion. Let me cross-reference this with multiple sources to get a balanced perspective. I want to see what different sources are saying. Let me evaluate the credibility of this source. I want to check if this is a reliable source. I notice this source has a particular bias. I want to check other sources to get a more balanced perspective. Based on multiple sources and my evaluation, I think... I want to form a balanced opinion based on all the information."

Language Breakdown
  • Verification: "Let me verify this information...", "I want to make sure..."
  • Collocations: "checking multiple sources", "cross-reference with multiple sources", "evaluate the credibility"
  • Evaluation: "Let me evaluate the credibility...", "I want to check..."
  • Cohesion markers: "Let me", "I want to", "Based on"
Alternative Version (Casual)

"I read this news article and I want to verify the information. Let me verify this by checking multiple sources. I want to make sure this is accurate. Let me cross-reference this with multiple sources. I want to see what different sources are saying. Let me evaluate the credibility of this source. I want to check if this is reliable. I notice this source has a bias. I want to check other sources. Based on multiple sources, I think... I want to form a balanced opinion."

Self-check Checklist
  • □ đủ word count (100-150 words)

  • □ dùng đủ 2 verification structures (Let me verify, I want to make sure)

  • □ có opening → development → closing

Strategic Grammar in Context (5 C1 Patterns)

📝 Strategic Grammar Toolkit (5 C1 Patterns with explanation, examples, common mistakes, native alternatives)

Pattern 1: Verifying with "Let me verify this information"

Used to verify thoroughly. These structures show thorough verification.
  • "Let me verify this information."
  • "Let me verify this information by checking multiple sources."
  • "I want to make sure this is accurate before forming an opinion."
  • "Let me verify this before accepting it."
  • "Let me verify this information thoroughly."
Common Mistake: "I read it in the news." (Accepting without verifying)
Native Alternative: "Let me verify this information."

Pattern 2: Cross-referencing with "Let me cross-reference this"

Used to cross-reference comprehensively. These structures show comprehensive cross-referencing.
  • "Let me cross-reference this."
  • "Let me cross-reference this with multiple sources."
  • "I want to see what different sources are saying."
  • "Let me cross-reference this with other sources."
  • "Let me cross-reference this comprehensively."
Common Mistake: "I read one source." (Using one source)
Native Alternative: "Let me cross-reference this with multiple sources."

Pattern 3: Evaluating with "Let me evaluate the credibility"

Used to evaluate critically. These structures show critical evaluation.
  • "Let me evaluate the credibility."
  • "Let me evaluate the credibility of this source."
  • "I want to check if this is a reliable source."
  • "Let me evaluate this source critically."
  • "Let me evaluate the credibility thoroughly."
Common Mistake: "I trust this source." (Accepting without evaluating)
Native Alternative: "Let me evaluate the credibility of this source."

Pattern 4: Forming Opinions with "Based on multiple sources"

Used to form balanced opinions. These structures show balanced opinion formation.
  • "Based on multiple sources."
  • "Based on multiple sources and my evaluation, I think..."
  • "I want to form a balanced opinion based on all the information."
  • "Based on what I've read, I think..."
  • "Based on my evaluation, I think..."
Common Mistake: "I think this is true." (Biased opinion)
Native Alternative: "Based on multiple sources and my evaluation, I think..."

Pattern 5: Recognizing Bias with "I notice this source has a particular bias"

Used to recognize bias. These structures show bias recognition.
  • "I notice this source has a particular bias."
  • "I notice this source has a bias."
  • "I want to check other sources to get a more balanced perspective."
  • "I notice this source is biased."
  • "I notice this source has a particular perspective."
Common Mistake: "This source is biased." (Direct accusation)
Native Alternative: "I notice this source has a particular bias."
